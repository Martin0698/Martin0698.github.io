Welcome to Alveus


These are the steps you have to follow to add your files (logs and configuration)

To add the files we must follow the following steps:

  1. Have the log file or log files to analyze.
  2. Have the JSON file with the configuration or add the configuration manually. (Example: JSON)
  3. Once we have the files ready, we click on the add files button and select the mode. (for manual mode we recommend that it only be used for a single service)
  4. After having uploaded the respective files or having added the manual configuration, we click on the button add files.
